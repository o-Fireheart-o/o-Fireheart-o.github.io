```html

        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge"
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First Web Page!</title>
    </head>

    <body>
        <!--웹 페이지에 표시할 내용을 적습니다.-->
        <p>나의 첫 번째 웹 페이지</p>
    </body>
</html>
```

### 1. 태그
웹 페이지를 구성하는 요소는 텍스트, 이미지, 버튼 등 다양하다.
**태그**는 이런 다양한 구성 요소를 정의하는 역할을 한다.

```html
<태그명>
```

### 2. 속성
**속성**은 태그에 어떤 의미나 기능을 보충하는 역할
태그 없이 단독으로 사용할 수는 없다.
속성은 속성명과 속성 값으로 구성

```html
<태그명 속성명="속성값">
```

### 3. 문법
#### 컨텐츠가 있는 문법
컨텐츠가 있는 문법에서는 컨텐츠 앞뒤를 태그로 감싼다.

```html
<title> My First Web Page! </title>
```

태그에는 시작 태그와 종료 태그가 있으며, 종료태그는 태그명 앞에 슬래시(/)를 넣어서 표현한다.
시작태그와 종료태그, 그리고 태그 사이의 컨텐츠를 합쳐서 **요소(Element)**라고 한다.

#### 컨텐츠가 없는 문법
컨텐츠가 없는 문법은 앞 뒤로 감싸야할 컨텐츠가 없으므로 시작 태그만 사용한다.
그래서 내용이 비어있다는 의미로 **빈 태그(empty tag)** 라고도 부른다.

```html
<br>
```

컨텐츠가 없는 문법은 시작 태그가 곧 요소여서 위 문법의 br 태그는 **br요소**라고 할 수 있다.
