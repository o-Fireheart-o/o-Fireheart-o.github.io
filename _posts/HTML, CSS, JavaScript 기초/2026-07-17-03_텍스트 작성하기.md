### 01. hn 태그
hn 태그는 제목이나 주제를 나타내는 텍스트를 표현할 때 사용

```html
<hn>제목</hn>
```

hn태그는 h1부터 h6까지 6개의 태그가 있는데, 여기서 h는 heading을 의미하고 n에들어가는 숫자는 중요도를 나타낸다.
즉 같은 hn태그라도 h1태그가 가장 중요하며 h6태그가 상대적으로 덜 중요하다.

> ~~그냥 옵시디언의 제목 태그와 비슷한 느낌이라고 이해하면 될 것 같다.~~
> 각 태그 번호를 순차적으로 사용해야 한다고 한다(중간에 번호를 건너 뛸 수 없다).
> 옵시디언의 제목과는 다르다.

hn태그로 작성된 텍스트는 검색 엔진에서 검색되는 키워드로 인식된다.

### 02. p 태그
p 태그는 본문의 문단(paragraph)를 작성할 때 사용한다.

```html
<p>내용</p>
```

HTML에서는 제목이나 주제를 나타내는 텍스트가 아니면 대부분 본문이기 때문에 p 태그를 자주 사용한다.

### 03. br 태그
br 태그는 문단에서 줄 바꿈할 때 사용

```html
<br>
```

활용 예시
```html
<p>
	안녕하세요,<br>
	코딩 공부하기 좋은 날입니다.
</p>
```

### 04. bloockquote 태그
blockquote 태그는 출처에서 인용한 문단 단위의 텍스트를 작성할 때 사용한다.
이때 출처가 확실한 인용문은 cite 속성으로 출처 경로를 명시해야 한다.

```html
<blockquote cite="출처 URL">문단 단위 인용문</blockquote>
```

### 05. q 태그
q 태그는 문단 안에 텍스트 단위의 짧은 인용문을 작성할 때 사용할 수 있는 태그이다.
q 태그를 사용한 콘텐츠는 큰따옴표("")로 묶인다.

```html
<q cite="출처 URL">짧은 인용문</q>
```

### 06. ins와 del 태그
ins 태그는 새로 추가된 텍스트임을 나타낼 때 사용하고, del 태그는 기존에 있던 텍스트가 삭제된 텍스트임을 나타낼 때 사용.
ins 태그의 컨텐츠에는 밑줄이
del 태그의 컨텐츠에는 취소선이 생긴다.

```html
<ins>추가 텍스트</ins>
<del>삭제 텍스트</del>
```

### sub와 sup 태그
sub 태그와 sup 태그는 각각 아래 첨자, 위 첨자에 해당하는 텍스트를 작성할 때 사용한다.

```html
<sub>아래 첨자</sub>
<sup>위 첨자</sup>
```
