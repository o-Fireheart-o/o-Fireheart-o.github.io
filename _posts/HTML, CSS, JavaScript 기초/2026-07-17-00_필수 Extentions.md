# Live Server / Ritwick Dey
![[Pasted image 20260717073145.png]]
Live Server는 가상 서버를 이용해, 작성한 HTML 코드를 실시간으로 확인할 수 있는 확장 프로그램.

# Auto Rename Tag / Jun Han
![[Pasted image 20260717073532.png]]
HTML 태그는 시작 태그와 종료태그로 구성되어 있다. 시작 태그와 종료태그는 항상 같은 태그명이어야 해서, Auto Rename Tag는 둘 중 한 쪽을 수정했을 때 다른 쪽은 자동으로 수정되게 설정 가능

# HTML to CSS autocompletion / solnurkarim
![[Pasted image 20260717073911.png]]
HTML 문서에서 class 속성으로 작성한 값을 연결된 CSS 파일에서 자동 완성되도록 지원

# HTML CSS support / ecmel
![[Pasted image 20260717074044.png]]
CSS 파일에서 작성한 클래스나 아이디 선택자 값을 연결된 HTML 문서의 id 속성과 class 속성값으로 적을 때 자동 완성 지원

# Code Runner / Jun Han
![[Pasted image 20260717074206.png]]
